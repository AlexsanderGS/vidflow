# 🎬 VidFlow

Projeto desenvolvido durante a formação Desenvolvedor Front-End da Alura, com foco no aprendizado do ambiente moderno de desenvolvimento front-end utilizando Node.js e terminal.

O projeto consiste em uma aplicação inspirada no YouTube, onde são exibidos vídeos em uma interface web, evoluindo de um projeto estático para um fluxo profissional utilizando ferramentas modernas do ecossistema JavaScript.
---

## 🚀 Objetivo do Projeto

O objetivo principal do curso foi aprender a configurar e utilizar o ambiente de desenvolvimento front-end moderno, incluindo:

- Utilização do Node.js
- Uso do terminal/CLI
- Gerenciamento de dependências com NPM
- Execução de scripts automatizados
- Utilização de bibliotecas externas
- Migração de projeto tradicional para Vite
- Deploy de aplicação em produção
---

## 🧠 Conceitos e Tecnologias Aprendidas

## 🛠️ Evolução do Projeto

O projeto já possuía inicialmente:

- ✅ HTML
- ✅ CSS
